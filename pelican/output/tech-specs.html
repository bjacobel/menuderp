Tech Specs

Tech Specs

This post is for those of you interested in how Menuwatch was built and runs.

Basics

Menuwatch is built on version 1.5 of Django, a Python web framework. The project is based on rdegges' Django skeleton, which is fantastic at eliminating the tedium of configuring a Django app to play nicely with addons like South and Celery, which Menuwatch makes heavy use of.

Front-end

Menuwatch uses the 3.0 RC of the Bootstrap framework as a helper to achieve fully responsive layout and cross-browser standardization. Its JavaScript utilizes jQuery 1.10.

Back-end

Menuwatch is hosted on Heroku using Gunicorn. Its database backend is Postgres. Menuwatch's static assets are hosted on S3 to reduce Heroku load, using the django-storages module.

Development

I develop on a MacBook Pro using Sublime Text 3 and Chrome Canary. I isolate my Python environment with virtualenvwrapper and occasionally use Sequel Pro for GUI SQL management. I swear by Homebrew.

Blogging

This blog is based on Pelican, a static site generator in Python. The theme is forked from wting's pelican-svbtle, inspired by Svbtle's interface. Hosting is done on Github Pages.

Speaking of GitHub…

… all of Menuwatch's code is Git-controlled and available on GitHub. Fork me.
